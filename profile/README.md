# Event Horizon Stories

Event Horizon Stories is a shared science-fiction universe spread across small, focused Elixir repositories.

Mars begins as a fragile runtime held together by procedure and repair. Then the distance widens. Probes leave for deep space and start making decisions before home can answer. Signals crowd the wire. Dispatch offices inherit promises they cannot afford to drop. Trade law hardens into bureaucracy. Wormholes force history to defend itself. Far beyond all of that, an inquiry engine studies the oldest surviving traces of reality and treats creation itself like a corrupted event log.

Across the organization, software is never separate from the conditions around it. Delay changes authority. Failure changes structure. Memory changes what counts as truth.

## Shared Conditions

The same pressures keep returning:

- distance
- delay
- coordination
- recordkeeping
- institutional memory
- conflicting truths
- histories that refuse to stay settled

Each repository isolates one part of that pressure and lets the system harden around it.
